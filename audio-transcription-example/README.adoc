[[audio-transcription-example]]
= Audio Transcription Example

This Spring AI example contains a simple Spring Boot application using Spring AI's https://docs.spring.io/spring-ai/reference/api/audio/transcriptions.html[Audi Transcription API] to transcribe an MP3 audio file to text.

By default, this example runs Spring AI with https://openai.com[OpenAI] using the https://platform.openai.com/docs/models/whisper-1[Whisper-1 model] to transcribe audio to text.

This example already includes a https://github.com/PacktPublishing/Building-Intelligent-Applications-With-Spring-AI/blob/main/audio-transcription-example/src/main/resources/audio.mp3[MP3 file] for running the program out-of-the-box.
