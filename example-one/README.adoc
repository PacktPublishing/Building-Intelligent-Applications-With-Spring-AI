[[example-one]]
= Example One

The first example from the book, "Building Intelligent Applications with Spring AI".

This example uses Spring AI with OpenAI to demonstrate how to use Spring AI's `ChatModel` to prompt ChatGPT
with a question and then receive the generated response, outputting the answer to standard out.

This example can be run with from the project base directory with `mvnw` command:

```sh
$ mvnw --projects example-one spring-boot:run
```
