[[example-one]]
= Example One

This is the first example from the book, "Building Intelligent Applications with Spring AI".

This example uses Spring AI with OpenAI to demonstrate how to use `ChatModel` API to prompt ChatGPT
with a question and receive a generated response, outputting the answer to standard out.

This example can be run from the project base directory with `mvnw` command:

```sh
$ mvnw --projects example-one spring-boot:run
```

This example currently support 3 different AI models: [ OpenAI, Google VertexAI Gemini, Ollama ]. Use the following Maven Profile IDs:

* openai (default)
* ollama
* gemini

To activate another AI model, such as Ollama (running locally), simply execute:

```sh
$ mvnw --projects example-one -P ollama spring-boot:run
```
